以下為專案內容：

import data
data("Boston")
df <- Boston

str(df)


hist(df$age, col = "green")


summary(df)

sample
# approach 1
nrow(df)
rows <- sample(1:nrow(df), nrow(df) * .8, replace = F)

train_base <- df[rows, ]
test_base <- df[-rows, ]

# check our data
dim(train_base)
dim(test_base)

# approach 2
rows2 <- createDataPartition(df$age, times = 1, p =.8, list = F)

training <- df[rows2, ]
testing <- df[-rows2, ]

# check our data
dim(training)
dim(testing)
Construct three different types of model
model <- function(method) {
  train(age ~ .,
        data = training,
        method = method,
        trControl = trainControl(method = "repeatedcv",
                                 number = 2,
                                 repeats = 2))
}

model_lm <- model(method = "lm")
model_rf <- model(method = "ranger")
model_gbm <- model(method = "gbm")
Compare model
sample <- resamples(list(Liner = model_lm,
                         Forest = model_rf,
                         GBM = model_gbm))
bwplot(sample)


dotplot(sample)


summary(sample)

 

補充：
  1. str() in R allows us to see the object type.
  2. RMSE stands for “Root Mean Squared Error”, which is used to evaluate the model’s performance.
  3. Random Forest is an ensemble learning method for classification and regression.
  4. GBM stands for “Gradient Boosting Model”
    • Understanding Gradient Boosting Machines
  5. The caret function “resamples()” compiles the model results and returns evaluation metrics.
